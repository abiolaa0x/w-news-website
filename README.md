## Overview

This project is a fully responsive news site layout designed to showcase latest news articles with a clean and professional UI. It features:

- A three-column grid layout for desktop, rearranged into a single-column layout for mobile.
- Hero section with large images and compelling headlines.
- Sidebar with "New" articles section.
- Trending articles section displayed in a card style.
- A mobile-friendly hamburger menu with smooth open/close animations.
- Color scheme and typography inspired by a simple style guide, using CSS variables for maintainability.

---

## Features

- **Responsive design**: Seamless adaptation from desktop to mobile devices.
- **CSS Grid and Flexbox**: Modern CSS techniques for layout and alignment.
- **Mobile hamburger menu**: Hidden desktop menu replaced with a toggleable mobile menu.
- **Clean and accessible code**: Semantic HTML, clear class names, and keyboard accessible controls.
- **Custom CSS variables**: Easy theming and color customization.

---

## Technologies Used

- **HTML5**
- **CSS3** (Grid, Flexbox, Variables, Transitions)
- **JavaScript** (Vanilla for hamburger toggle)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/w-news-website.git

   ```

2. Navigate to the project Folder:

   ```bash
   cd w-news-website
   ```

3. Open index.html in your preferred browser.

## Usuage

Open the website in any browser.

- On desktop, navigation links are displayed in the top bar.
- On mobile or smaller screens:
  - Click the hamburger icon (☰) to open the menu.
  - Click the close (×) button or overlay to close the menu.
- Hover over links and article titles to see smooth color changes.
- Explore the hero section, new articles, and trending articles — all sections adjust responsively.
